In a `tabulae` dataset, the directory `rules-tables` contains the rules for applying endings to stems in your lexicon.  You can add `.cex` files to the following eight subdirectories:




-   adjectives
-   adverbs
-   gerundives
-   gerunds
-   infinitives
-   participles
-   supines
-   verbs (conjugated forms)
